# react-native-expo-template


# Installation

<ol>
  <li>yarn global add expo-cli - installs the  expo-cli globally</li>
  <li>yarn/expo install - installs project deps</li>
</ol>

# Scripts

<ol>
  <li>expo start / yarn start - starts the metro server</li>
  <li>yarn lint - runs the linter</li>
  <li>yarn test - runs test scripts</li>
</ol>

# Setup
  ### Android
  
<ul>
  <li>Make sure to have a emulator installed. You may want to look at [Genymotion](https://www.genymotion.com/) or [Android Studio AVD Manager](https://developer.android.com/studio/run/managing-avds) </li>
    <li> Make sure to add the following in your env variables
    <ol>
    	<li>Android/sdk/platform-tools (Android sdk)</li>
       	<li>JRE_HOME/JAVA_HOME (JRE or jdk)</li>
    </ol>
    </li>
</ul>

### iOS
<ul>
	<li>Just make sure that you have iOS simulator and using an apple machine. It just works!</li>
</ul>
